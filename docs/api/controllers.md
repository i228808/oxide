# API: Controllers

Reference for controller surface from:

- `oxide-framework-core/src/controller.rs`
- `oxide-framework-macros/src/controller.rs`

## `Controller` Trait

Generated by `#[controller("/prefix")]` or implementable manually:

- `const PREFIX: &'static str`
- `fn from_state(state: &AppState) -> Self`
- `fn register(self: Arc<Self>) -> OxideRouter`
- `fn configure_router(router: axum::Router) -> axum::Router` (default pass-through)

## Macro Route Attributes

Supported in controller impl blocks:

- `#[get("...")]`
- `#[post("...")]`
- `#[put("...")]`
- `#[delete("...")]`
- `#[patch("...")]`
- `#[head("...")]`
- `#[options("...")]`

## Constructor Resolution

- If `fn new(state: &AppState) -> Self` exists, it is used.
- Otherwise generated code falls back to `Default::default()`.

## Controller Middleware Hook

Optional in impl block:

```rust
fn middleware(router: axum::Router) -> axum::Router
```

When present, it is wired through `configure_router` and applies only to that
controller’s routes.
