# API Reference

This page is the canonical top-level API map for `oxide_framework_core`.
It is derived from `oxide-framework-core/src/lib.rs`.

## Core Types

| Export | What it is | Defined in |
|---|---|---|
| `App` | Main application builder and runtime entry point | `app.rs` |
| `TestServer` | Test helper returned by `App::into_test_server()` | `app.rs` |
| `AppConfig` | YAML + env configuration model | `config.rs` |
| `AppState` | Shared runtime state container | `state.rs` |
| `Controller` | Trait generated by `#[controller]` | `controller.rs` |
| `OxideRouter` | Composable routing wrapper around Axum | `router.rs` |
| `Method` | HTTP method enum for generic registration | `router.rs` |
| `ApiResponse<T>` | Standard JSON response envelope | `response.rs` |

## Extractors

| Export | Type | Purpose |
|---|---|---|
| `Config` | extractor | Access `Arc<AppConfig>` in handlers |
| `Data<T>` | extractor | Access app singleton state (`Arc<T>`) |
| `Inject<T>` | extractor | Alias of `Data<T>` for readable controller signatures |
| `Scoped<T>` | extractor | Access request-scoped dependency created by `App::scoped_state(...)` |

## Auth API

| Export | Purpose |
|---|---|
| `AuthConfig` | Configure JWT validation (secret, issuer/audience, bearer/cookie sources) |
| `AuthClaims` | Decoded JWT claims inserted into request extensions |
| `AuthLayer` | Auth middleware layer used by `App::auth(...)` |
| `Authenticated` | Extractor that requires a valid identity |
| `OptionalAuth` | Extractor that allows anonymous requests |
| `RequireRole<R>` | Extractor enforcing a role guard |
| `RoleName` | Trait mapping marker type to role string |
| `AuthRejection` | Rejection for auth extractors |
| `encode_token` | Utility for minting HS256 JWTs |

## Axum Re-exports
